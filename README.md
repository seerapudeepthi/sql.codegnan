# Banking System + Fraud Detection

## Project Overview
The Banking System + Fraud Detection is a MySQL database project designed to manage banking operations and identify suspicious transactions using SQL queries. It demonstrates relational database design, table relationships, and data analysis through real-world banking scenarios.

---

## Technologies Used
- MySQL
- SQL

---

## Database Tables

The project contains the following tables:

1. Customers
2. Accounts
3. Branches
4. Employees
5. Transactions
6. Loans
7. Cards
8. Fraud_Alerts

---

## Relationships

- One customer can own multiple accounts.
- One branch manages multiple accounts and employees.
- One account can have multiple transactions.
- One customer can apply for multiple loans.
- One account can be linked to multiple cards.
- Fraud alerts are generated for suspicious transactions.
